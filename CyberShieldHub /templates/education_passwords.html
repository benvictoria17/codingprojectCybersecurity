Learn About Password Security

What is Password Security?

Password security means using strong passwords and keeping them safe to stop hackers from getting into your accounts.

Just like you need a special key to get into your house, passwords are the keys to your online accounts. If a bad person gets your password, they can pretend to be you!

Why are Strong Passwords Important?

  • Protect your personal information
  • Stop hackers from taking over your accounts
  • Keep your messages and photos private
  • Prevent others from pretending to be you
  • Protect accounts that have payment information

Weak Passwords You Should Never Use

Hackers can easily guess these common passwords. Never use them!

Very Weak Passwords

  • password
  • 123456
  • qwerty
  • abc123
  • admin
  • welcome

Personal Information (Never Use)

  • Your name
  • Your birthday
  • Your pet's name
  • Your favorite team
  • Your phone number
  • Your address
Did you know? Hackers can try thousands of common passwords per second using special software. They also use information from your social media to guess passwords!

How to Create Strong Passwords

Make it Long

Use at least 12 characters. Longer passwords are harder to crack.

Example: sunshine → sunshine-mountain-bicycle

Mix it Up

Use a mix of uppercase letters, lowercase letters, numbers, and symbols.

Example: dolphin → D0lph!n_2023

Use a Passphrase

Use a series of random words with spaces or symbols between them.

Example: correct-horse-battery-staple

Different Passwords for Different Sites

Use a different password for each website or app, especially important ones like email and banking.

Email: S@fe-Em@il-Account!2023

Gaming: G@m!ng-F0r-Fun-2023

How Passwords Work: The Technical Side

Let's look at how websites store and check your passwords:

Websites Don't Store Your Actual Password

Good websites don't keep your real password. Instead, they turn it into a scrambled code called a "hash." When you log in, they check if your password matches the hash.

Here's how a website stores your password in JavaScript:
// This is a simplified example of how passwords are hashed
function hashPassword(password) {
  // In real websites, they use much stronger methods than this
  let hashedPassword = '';
  
  // This creates a special code from your password
  for (let i = 0; i < password.length; i++) {
    hashedPassword += password.charCodeAt(i).toString(16);
  }
  
  return hashedPassword;
}

// Example
const userPassword = "Str0ng_P@ssw0rd!";
const storedHash = hashPassword(userPassword);

console.log("Your password: " + userPassword);
console.log("What the website stores: " + storedHash);

// When you try to log in later:
function checkLogin(enteredPassword, storedHash) {
  const newHash = hashPassword(enteredPassword);
  
  if (newHash === storedHash) {
    console.log("Login successful!");
    return true;
  } else {
    console.log("Wrong password!");
    return false;
  }
}

Important to Know!

Even if a website gets hacked, hackers can't easily figure out your actual password if the website used good hashing. This is why using different passwords for different websites is important!

How Hackers Try to Crack Passwords

Attack Type How It Works How to Protect Yourself
Dictionary Attack Hackers try every word in the dictionary Don't use single dictionary words as passwords
Brute Force Attack Hackers try every possible combination of characters Use long passwords with different character types
Rainbow Table Attack Hackers use pre-computed tables of password hashes Use passwords with special characters and unique combinations
Social Engineering Hackers trick you into telling them your password Never share your password with anyone, even if they say they're from the company

Password Manager: A Great Solution

Password managers are special programs that:

  • Generate super strong passwords for you
  • Remember all your passwords so you don't have to
  • Fill in your passwords automatically on websites
  • Keep your passwords encrypted (securely scrambled)
  • Only require you to remember one master password

Many password managers are free to use, and they work on phones and computers!
